# Twelve Stones: Ephod Quest - Marketing Website

This folder contains the marketing website for Twelve Stones: Ephod Quest, a wholesome biblical adventure game.

## Overview

The website is built with:

- **HTML5** for structure
- **CSS3** with custom properties for the Earth & Gold color palette
- **Vanilla JavaScript** for interactivity
- **Responsive design** for mobile-first experience

## Features

- **Hero Section**: Eye-catching introduction with key stats
- **Story Section**: Biblical context and game narrative
- **Features Section**: Game mechanics and unique selling points
- **Gallery Section**: Screenshots with modal viewer
- **Download Section**: Platform-specific download buttons
- **About Section**: Developer information and mission
- **Responsive Navigation**: Mobile-friendly hamburger menu
- **Smooth Scrolling**: Enhanced user experience
- **Image Gallery**: Click-to-enlarge functionality

## Color Palette

The website uses the same Earth & Gold palette as the game:

- Desert Sand: `#E9D5B3`
- Clay Brown: `#8B6F47`
- Olive Green: `#6B7A4F`
- Pure Gold: `#FFD700`
- Antique Gold: `#C5A572`

## Deployment

1. **Local Development**: Open `index.html` in any modern web browser
2. **Web Server**: Upload all files to your web hosting service
3. **CDN**: Consider using a CDN for faster image loading
4. **Analytics**: Add Google Analytics or similar tracking code
5. **Domain**: Point your domain (ephodquest.com) to the hosting

## Customization

### Colors

Edit the CSS custom properties in `:root` to change the color scheme.

### Content

Update text content in `index.html` to reflect current game status, download links, etc.

### Features

Add new sections or modify existing ones by editing the HTML structure and corresponding CSS.
